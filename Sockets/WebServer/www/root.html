• /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /github?query=users/amehlhase316/repos -- returns public repositories of user amehlhase316 on GitHub
  • /multiply?num1={number}&num2={number} -- multiplies two numbers provided as parameters
  • /roll?sides={number}&numDice={number} -- simulates rolling dice with the specified number of sides per dice and the number of dice
  • /temperature?value={number}&from={unit}&to={unit} -- converts temperature from one unit to another (C, F, or K)

Examples:

  • /file/sample.html - View the content of sample.html file
  • /json - View JSON data of random request
  • /random - View index.html
  • /github?query=users/amehlhase316/repos - View public repositories of user amehlhase316 on GitHub
  • /multiply?num1=12&num2=12 - Multiply two numbers (12 and 12)
  • /roll?sides=6&numDice=3 - Simulate rolling 3 six-sided dice
  • /temperature?value=90&from=F&to=C - Convert 90 F to C
